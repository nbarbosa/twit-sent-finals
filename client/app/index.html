This is a final project (right?) for the Natural Language Processing (NLP) class at Syracuse University.
I'm using a Maximum Entropy classifier with 80.5% accuracy utilizing distant supervision and some manually annotated data.
The training dataset is balanced and has 1,618,504 tweets labeled either positive or negative.
